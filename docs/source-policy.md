# Source Policy

ConfAI publishes only source-grounded conference facts.

## Allowed Sources

Preferred source hierarchy:

1. Official conference website.
2. Official CFP page.
3. Organizing society or university page.
4. Publisher/proceedings page.
5. Reputable academic calendars as discovery leads only.

Polish-language sources are allowed only for Poland.

## Excluded Sources

- AI-generated listings.
- Unconfirmed sources.
- Blank or broken links.
- Paywalled, login-only, CAPTCHA-protected, or explicitly anti-scraping pages.
- Sources where robots.txt or terms prohibit the intended access.

## Publication Rules

Every published record needs:

- source URL;
- `last_checked_at`;
- confidence level;
- review status;
- explicit missing values where data is unavailable.

Low-confidence or conflicting records go to review instead of being published.
